## Setup and Run Instructions

To run this Flask Hello World application:

1. Create a virtual environment (recommended):
   ```
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Run the Flask application:
   ```
   python app.py
   ```

4. Open your web browser and navigate to http://127.0.0.1:5000/

5. Enter your name in the input field and click the "Greet Me" button to see a personalized hello message.

The application structure includes:
- Flask backend with a REST API endpoint
- HTML template for the user interface
- CSS for styling
- JavaScript for handling user interactions

All files are organized in the standard Flask project structure with static files and templates in their respective directories.
