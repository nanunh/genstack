## Setup and Run Instructions

To set up and run the Hindu Calendar application:

1. Make sure you have Python 3.6+ installed on your system.

2. Create a new directory for the project and navigate to it in your terminal.

3. Create the file structure as shown in the project files.

4. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

5. Run the Flask application:
   ```
   python app.py
   ```

6. Open your web browser and navigate to:
   ```
   http://127.0.0.1:5000/
   ```

7. You should now see the Hindu Calendar application with the current month displayed. You can navigate between months using the "Previous Month" and "Next Month" buttons.

8. Click on any day to see detailed Rahu Kalam and Yamakandam timings for that day.

The application uses traditional fixed time periods for Rahu Kalam and Yamakandam based on the day of the week. These timings are displayed for each day in the calendar.
