# PDF Merger Application - Setup Instructions

## Quick Start

1. **Install Python dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the application:**
   ```bash
   python app.py
   ```

3. **Open browser:**
   Navigate to `http://localhost:5000`

## How to Use

1. Upload 2 or more PDF files (drag & drop or browse)
2. Review the file list
3. Click "Merge PDFs"
4. Download your merged PDF

## Features

✅ Simple HTML/CSS/JS frontend
✅ Python Flask backend
✅ Drag and drop file upload
✅ Multiple PDF merge
✅ Instant download
✅ Modern responsive UI
✅ No database required
✅ Session-based file handling

## Technology Stack (As Requested)

- **Frontend**: Pure HTML, CSS, JavaScript (no frameworks)
- **Backend**: Python Flask
- **PDF Processing**: PyPDF2

The application is production-ready with error handling, file validation, and automatic cleanup!
