# Employee Assignment Tracker - Setup Instructions

## Getting Started

1. Download all the files from this project:
   - index.html
   - styles.css
   - app.js

2. Keep all files in the same directory/folder.

3. Open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, or Edge).

## Using the Application

### Adding Assignments
- Fill out the form on the left side with employee details and assignment information
- Click "Add Assignment" to create a new work assignment

### Viewing Assignments
- All assignments appear as cards in the main section
- Use the tabs (All, Hardware, Software) to filter by assignment type
- Use the filter controls to search by employee name, type, or status

### Managing Assignments
- Click "Edit" to modify an existing assignment
- Click "Update Status" to quickly change just the status
- Click "Delete" to remove an assignment

## Data Storage

All data is stored in your browser's localStorage, which means:
- Your assignments will persist even if you close the browser
- The data is stored only on your device
- If you clear browser data/cache, your assignments will be deleted

## No Server Required

This is a client-side only application that runs entirely in your browser without needing any server setup.
